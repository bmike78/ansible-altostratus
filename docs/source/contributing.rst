Contributing
============

Altostratus is an Open Source project and welcomes contributions, whether they are in
the form of feedback, comments, suggestions, bugs, code contributions or
code reviews.

Altostratus uses GitHub for issues or pull requests.

Filing issues and bugs
----------------------
Please contribute!

You can really make a difference by:
`Making an issue`_. A well described issue helps a lot.

`Making a pull request`_ when you see the error in code.

.. _Making an issue: https://help.github.com/articles/creating-an-issue
.. _Making a pull request: https://services.github.com/on-demand/github-cli/open-pull-request-github
